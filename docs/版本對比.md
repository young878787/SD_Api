## API 架構對比

### 原版 (SD WebUI)

```python
# 1. 準備配置
config = {
    "prompt": full_prompt,
    "negative_prompt": "bad quality...",
    "steps": 20,
    "sampler_name": "Euler a",
    "width": 512,
    "height": 768,
    "seed": random.randint(...)
}

# 2. 發送請求（同步）
response = requests.post(
    f"{sd_url}/sdapi/v1/txt2img",
    json=config,
    timeout=300
)

# 3. 獲取圖片
images = response.json()['images']
for img_base64 in images:
    image = Image.open(io.BytesIO(base64.b64decode(img_base64)))
    image.save(filepath)
```

### 新版 (ComfyUI)

```python
# 1. 載入工作流
workflow = load_workflow_from_json("qwen image (1).json")

# 2. 修改提示詞
workflow["6"]["inputs"]["text"] = character_design
workflow["3"]["inputs"]["seed"] = random.randint(...)

# 3. 提交工作流（異步）
client = ComfyUIClient("127.0.0.1:8188")
ws = websocket.connect(f"ws://{server}/ws?clientId={client_id}")
prompt_id = queue_prompt(workflow)

# 4. 監聽執行狀態
while True:
    message = json.loads(ws.recv())
    if message['type'] == 'executing' and message['data']['node'] is None:
        break  # 完成

# 5. 獲取圖片
history = get_history(prompt_id)
for image_info in history['outputs'][node_id]['images']:
    image_data = get_image(image_info['filename'], ...)
    image = Image.open(io.BytesIO(image_data))
    image.save(filepath)
```

**原版**:

    def get_history(prompt_id)         # GET /history
    def get_images(prompt)             # WebSocket 監聽

class AutoCharacterGenerator:
    def __init__(self, comfyui_address, output_dir)

    def save_images()
    def run()

## 適用場景

### 原版適合

- ✅ 快速上手，不想學習 ComfyUI
- ✅ 只使用 SD 1.5/SDXL 模型
- ✅ 簡單的文生圖需求
- ✅ 不需要複雜的節點組合

### 新版適合

- ✅ 已經在使用 ComfyUI

## 遷移建議

### 從原版遷移到新版

1. **安裝 ComfyUI**
   ```bash
   git clone https://github.com/comfyanonymous/ComfyUI
   cd ComfyUI
   python main.py
   ```

2. **安裝新依賴**
   ```bash
   pip install websocket-client
   ```

3. **準備工作流**
   - 在 ComfyUI 界面設計工作流
   - 導出為 JSON (Save API Format)
   - 放到 `Api/` 目錄

4. **測試連接**
   ```bash
   python test_comfyui.py
   ```

5. **運行新程式**
   ```bash
   python auto_character_generator_comfyui.py
   ```

### 保留原版的情況

如果以下情況，建議保留原版：

- ❌ 不想學習 ComfyUI
- ❌ 原版運行穩定，不需要新功能
- ❌ 不需要使用特殊模型
